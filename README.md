# Task Manager Api

An express powered node.js application that provides a rest api for performing crud operations on tasks and supports user authentication as well as uploading user profile.

## Usage
To run the project locally, follow these steps below:
* Install MongoDB database on your computer [if not already installed](https://www.mongodb.com/try/download/community).
* Install [Node.js](https://nodejs.org/en/download/) if not already installed.
* Clone this repository, create a config folder in the root folder and create a .env.dev file, add the enviromental variable **MONGODB_URL**, the database uri for mongodb.
* Install the dependencies by running ```npm install``` and run the command ```npm run dev``` to start up your node.js application.

## Technologies
Project was created with:
* [Node.js](https://nodejs.org/)
* [Express](https://expressjs.com/)
* [MongoDB](https://www.mongodb.com)
